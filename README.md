# honkipass 5

Sample: <https://michinobu.jp/honkipass/>

## Development

### Prerequisites

- nvm or node >=18

### Build

```bash
git clone git@github.com:MichinobuMaeda/honkipass5.git
cd honkipass5
npm install
npm run build
```

### Tools used

- [Material Theme Builder](https://www.figma.com/community/plugin/1034969338659738588/material-theme-builder)
